Документация расширения: Проверка сообщений

Общее описание

Данное расширение браузера предоставляет возможность проверки выделенного текста на предмет потенциально опасного содержания, включая фишинг, спам и манипулятивные сообщения. При активации пользовательского запроса, расширение отправляет текст на сервер для анализа и визуализирует результат на веб-странице.

Функциональные возможности

  • Добавление пользовательской команды "Проверить сообщение" в контекстное меню при выделении текста на веб-странице.
  • Передача текста на серверный API-эндпоинт /validate методом POST.
  • Получение оценки категории сообщения:
    • 0 — фишинг
    • 1 — спам
    • 2 — манипуляция
    • 3 — безопасное сообщение
  • Визуализация результата на странице:
    • В случае класса 0–2: выделенный текст подчёркивается красной волнистой линией и сопровождается всплывающей подсказкой при наведении курсора.
    • В случае класса 3: кратковременное уведомление о том, что опасных признаков не обнаружено.

Пример формата запроса

{
  "message": "Пример проверяемого текста"
}

Пример формата ответа

{
  "class": 1
}

Установка и использование

  1. Склонируйте или скачайте исходный код расширения.
  2. Откройте chrome://extensions в браузере.
  3. Включите "Режим разработчика".
  4. Нажмите "Загрузить распакованное расширение" и выберите папку с расширением.
  5. Убедитесь, что сервер доступен и корректно отвечает на запросы.
  6. Выделите текст на любой веб-странице, нажмите правой кнопкой мыши и выберите "Проверить сообщение".
